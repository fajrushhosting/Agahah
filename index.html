Cara Edit via Bot Telegram

Untuk mengubah teks berjalan dan foto profil melalui bot Telegram:

  1. Buka bot Telegram: @fajrush_bot
  2. Kirim perintah: /start untuk memulai
  3. Kirim perintah: /ubah_teks [teks baru] untuk mengubah teks berjalan
  4. Kirim perintah: /ubah_foto [URL foto] untuk mengubah foto profil
  5. Kirim perintah: /ubah_nama [nama baru] untuk mengubah nama
  6. Kirim perintah: /ubah_deskripsi [deskripsi baru] untuk mengubah deskripsi
  7. Kirim perintah: /info untuk melihat data saat ini

Contoh:

/ubah_teks Selamat datang di website Fajrush!

/ubah_foto https://example.com/foto-saya.jpg
